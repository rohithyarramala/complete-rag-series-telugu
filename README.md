## 🛠️ How to use this Repository (ఎలా ఉపయోగించాలి?)

1. **Open Colab:** ప్రతి ఫోల్డర్ లో ఉన్న `.ipynb` ఫైల్ పై క్లిక్ చేసి, పైన కనిపించే **"Open in Colab"** బటన్ నొక్కండి.
2. **Set API Key:** గూగుల్ కొలాబ్ లో ఎడమ వైపు ఉన్న **Secrets (🔑 key icon)** క్లిక్ చేసి `OPENAI_API_KEY` ని యాడ్ చేయండి.
3. **Upload Files:** ఏదైనా ఒక వీడియోలో `sample.txt` లేదా `PDF` ఫైల్స్ వాడితే, వాటిని మొదట కొలాబ్ ఎన్విరాన్మెంట్ లోకి అప్‌లోడ్ చేయండి.
